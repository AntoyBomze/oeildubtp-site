# OEILDUBTP

Site vitrine pour une offre de sécurisation de chantiers 24/7 avec caméras 4G autonomes solaires.

## Mise en ligne

Le site principal est dans `index.html`.

Pour publier avec GitHub Pages :
- ouvrir les paramètres du dépôt
- aller dans **Pages**
- choisir la branche `main`
- choisir le dossier `/root`
- enregistrer

L'URL de publication sera ensuite proposée par GitHub.
